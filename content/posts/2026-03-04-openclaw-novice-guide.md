## 为什么你的龙虾在"装死"？

很多人装完 OpenClaw，上来就问"你能干嘛""帮我写个东西"，但你连你是谁、你要干嘛、有什么喜好都没告诉它，它怎么可能给你想要的答案？

**核心问题：你没有做"岗前培训"。**

OpenClaw 最牛的核心能力是**永久记忆**。说白了，就是给你的龙虾做「岗前培训」，让它知道自己跟着谁干活，老板 的喜好、习惯、工作节奏是什么，以后干活才能精准踩在你的点子上。

### 小白必学的岗前培训

直接把下面这段信息发给龙虾就行，结尾一定要加一句：**以上内容请永久记住，这是我的基本个人信息。**

```
我叫 XX，今年 XX 岁，在 XX 城市做 XX 工作，主要负责 XX 内容。我平时不喜欢太正式的说话方式，你跟我沟通可以随意一点。我每天 XX 点上班，平时 XX 点睡觉。我目前在 XX / 有 XX 习惯，相关内容麻烦多提醒我
```

就这么几句话，不用写小作文，不用很正式。你还可以随时补充信息，今天告诉它你喜欢健身，明天告诉它你在学法语，这些内容它都会一一记下来，越用越懂你。

## 技能 vs 插件：90% 的人都搞混了

先给小白一句话讲透：

- **skill（技能）**：龙虾能学会的本事 —— 比如帮你搜网页、发邮件、整理文档、控制浏览器
- **插件**：给龙虾加个外设 —— 比如加个语音输入框、换个皮肤

新手完全不用先碰插件，先把龙虾的本事练起来，再搞那些花里胡哨的。

## 技能安装三大渠道

### 渠道 1：ClawHub 官方市场（新手闭眼冲）

官方亲儿子，OpenClaw 里直接就能进，一键安装，有官方安全扫描，不用怕踩坑。

### 渠道 2：GitHub 开源社区（爱折腾的直接冲）

这里全是开发者大佬自发做的技能，功能更全、更细分。特别推荐懒人宝藏仓库：**awesome-openclaw-skills**，从官方 13000+ 技能里筛选出 5400+ 优质技能。

### 渠道 3：水产市场（中文用户专属宝藏）

中文圈最火的社区，全是针对国人使用场景做的技能，比如飞书、微信、B 站这些，比海外平台接地气太多。

## 小白必装技能清单

### 核心必备（装了立刻见效）

1. **Capability Evolver** — 官方市场下载量第一，让龙虾越用越顺手，分析你的使用习惯自动优化回复方式
2. **Self-Improving Agent** — 社区口碑第一，让龙虾自复盘进化，每次干完活都会自己复盘哪里能优化
3. **Summarize** — 文档摘要神器，不管合同、论文、网页，10 秒提炼核心要点
4. **Find Skills** — 懒人技能导航仪，直接说"帮我找个能 XX 的技能"就搞定
5. **Agent Browser** — 给龙虾装上手和眼睛，能像人一样操作浏览器

### 场景刚需（按需取用）

- 打工人的飞书 / Notion / 企业微信 → 搜对应技能一键打通
- 开发者 → GitHub Skill，帮你改代码、审 PR、管 issue
- 日常提醒 → Weather，不用 API 密钥就能自动查天气

## 进阶组合：能力翻倍

### OpenClaw + Claude Code = 最强代码搭档

Claude Code 是专门写代码的，但记不住你的偏好；OpenClaw 刚好补上这个坑，记得你所有的需求、项目历史。

### OpenClaw + n8n = 自动化神器

n8n 能打通所有软件，但单独用要自己拖拽配置。组合后，你只用说话，它就能帮你自动跑流程。

### OpenClaw + Obsidian = 你的第二大脑

聊天的所有灵感、想法、干货，自动帮你整理进知识库，还能跟之前的笔记做关联。

### OpenClaw + Telegram = 随时随地跟着你

之前只能坐在电脑前用，打通 Telegram 之后，手机随时都能跟它说话。

## 写在最后

很多人觉得 OpenClaw 很难，其实根本不是。

装完只是第一步，是要你慢慢磨合、慢慢培养的打工仔。不用一开始就搞复杂玩法，先从最小的需求开始：

- 今天让它提醒你天气
- 明天让它帮你整理会议纪要
- 后天让它帮你处理邮件

慢慢你会发现，它越来越懂你，你也越来越会用。

不用怕自己是新手，所有的熟手，都是从"让龙虾先跑起来"这一步开始的。
